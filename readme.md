# POC NodeJS Express

POC Projekt to try NodeJS with Express.

# Development

## Start application

```sh
npm run start
```

## Api Calls

```sh
curl http://localhost:3000/todos
```

```sh
curl -X POST -H 'content-type:application/json' -d '{"description":"Test 1"}' http://localhost:3000/note-todo
```
